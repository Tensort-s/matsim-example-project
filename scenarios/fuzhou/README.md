# Fuzhou scenario configs

The root of this directory keeps only the current runnable Fuzhou simulation chain:

1. `config-transit-mode-choice-2pct-transferwait-metro40-cont10.xml`
2. `config-transit-mode-choice-2pct-waitpenalty-metroprefer-cont20.xml`
3. `config-transit-mode-choice-2pct-waitpenalty-metroprefer-from-cont20-reroute50.xml`

Older experiment configs are kept in `archive/` for provenance. Many archived configs refer to data or output paths that were intentionally cleaned up, so they should be treated as historical records rather than ready-to-run configs.

Current run outputs are stored under:

```text
runs/fuzhou/outputs/
```
