# Fuzhou city package

This directory describes the current single-city Fuzhou MATSim package. The heavy data remain under the stable first-level `data/` domains, with `fuzhou/` added as the city layer.

Key files:

- `city.yaml` records the active data roots, CRS, current demand, current transit supply, and final output.
- `../../runs/fuzhou/run_manifest.json` records the retained simulation run chain.
- `../../docs/DATA_LAYOUT_MIGRATION_FUZHOU.md` records the old-to-new path mapping.
